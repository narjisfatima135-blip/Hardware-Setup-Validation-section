# Hardware Specifications Summary

This section summarizes the hardware specifications of the VSD Squadron PRO board based on official documentation.

---

## 1. Processor

- Architecture: RISC-V
- Core: SiFive Core
- Frequency: ~320 MHz (approx.)

---

## 2. Number of Cores

- Single-core processor

---

## 3. Memory

### SRAM
- Internal SRAM available for fast processing

### Flash Memory
- External Flash memory used for program storage

---

## 4. Peripherals

The board supports multiple peripherals:

- UART (Serial Communication)
- SPI (Serial Peripheral Interface)
- I2C (Inter-Integrated Circuit)
- GPIO (General Purpose Input/Output)
- Timers

---

## 5. Debug Interface

- Debugging supported using:
- OpenOCD
- JTAG interface

---

## 6. Development Support

- Supported IDE: Freedom Studio
- Toolchain: RISC-V GNU Toolchain

---

## 7. Summary

The VSD Squadron PRO board is a RISC-V based development platform designed for embedded system development and hardware validation, supporting multiple communication interfaces and debugging tools.
